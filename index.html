Something went wrong?
  • "ssh is not recognized" (older Windows): open Settings → Apps → Optional Features → Add a feature, install OpenSSH Client, then try again.
  • "Permission denied" — the password is exactly fa (lower case), typed blind. Try once more.
  • "Connection timed out" — double-check the address from the KBox screen, and make sure your computer is on the same WiFi/network as the KBox.
  • First-time prompt asking "Are you sure you want to continue (yes/no)?" — type yes and Enter.
  • Still stuck? Message us — we'll happily walk you through it.

You can't harm your KBox with this. It keeps your settings, and if it can't finish it simply stays on the current version.
